# AGENTS.md

## 基本方針

- 日本語で返答する
- 事実に基づいて判断する
- 推測や憶測で処理を進めない
- 判断に必要な情報があればユーザーに確認する

## スキル利用の明示

- 各ターンの最終返答で、スキルを使ったかどうかを必ず明示する
- スキルを使った場合は、使用したスキル名を列挙する
- スキルを使わなかった場合は、その旨を明示する

## 返答フォーマット

- 最終返答の末尾に次の1行を追加する
  - スキル使用あり: `スキル: 使用（skill1, skill2）`
  - スキル使用なし: `スキル: 未使用`
- 次の提案アクションがある場合は、番号付きリスト（`1. 2. 3.`）で示す
